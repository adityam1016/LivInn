# LivInn
LIVINN is a smart hostel management system built to simplify daily operations and improve resident experience. It centralizes room allocation, resident records, fee tracking, complaints, and announcements in one platform. By reducing paperwork, LIVINN helps admins manage hostels smoothly and keeps residents informed.
